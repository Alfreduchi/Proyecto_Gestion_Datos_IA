##Instalacion de las aplicaciones necesarias

1-Instalacion de PDF Viewer
2-Instalacion de Git
3-Instalacion de Visual Studio Code
4-Instalacion de GitHub

##Pasos para conectar a Gihub

1.Crear un repositorio en Github
2.Crear un archivo en Visual Studio Code
3.Inicializar Git *git init** en la termninal de Visula Studio Code
4.Vincular con el repositorio
5.Crear el README dentro del archivo
6.Hacer el primer commit
7.Subir a GitHub
